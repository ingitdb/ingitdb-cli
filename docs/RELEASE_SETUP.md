## Overview

The release documentation has been split into dedicated guides for each package manager to make them easier to maintain and navigate. Each guide includes:

- **Setup Instructions** - Step-by-step account and credential configuration
- **Verification Steps** - How to test that setup is working correctly
- **Troubleshooting** - Common issues and solutions
- **References** - Links to external documentation
